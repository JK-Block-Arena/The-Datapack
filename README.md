## 如何使用

将本数据包（pvp3-JK.zip）放进方块竞技场存档的 datapacks 文件夹，删除 pvp3 文件夹，进入存档后在游戏中 `/reload` 再 `/function debug:init` 重置，可以使用 `/function debug` 打开神の面板。

[这里](https://www.bilibili.com/read/cv17401279/)也许会有一些修改指南。
