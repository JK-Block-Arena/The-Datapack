## 魔改内容
### 地图
**（暂时不放图片）**
- 大厅

有生涯数据统计查看，体验传奇连杀音效，旁观战场玩家方式，传送门走廊入口等。

- 传送门走廊

有进入赞助者大厅，创作者大厅，锻造&粒子大厅，1v1挑战入口。

- 沙滩

最早的原创地图，现在是第二个版本。

- 玄素

原本为官方弃案，现有爬墙玩法（非常驻地图）

- 瀑布

现在，是我的回合！

- 丛林

你可以看到花里胡哨的装饰!

- 商店

超大而且精美的商场，室内外都有战斗场所。

- 舞台

有传送门欸。

- 矿洞

除了容易迷路，挺好的。

### 突变
- 杀戮光环

攻击友方/敌方获得增益，但静息无法治疗

- 混沌灵魂++

职业仅能随机选择，也会因死亡或时间随机改变

- 一击必杀（不再常驻）

不计摔伤与死亡数，一击必杀你的对手吧！

- 你说我做

出现随机任务，完成与否奖惩分明

触发时，将从【短跑达人、和平模式、保持健康、蛙跳训练、就踩方块】中任选其一

- 黑暗森林

获得可静息的隐身，但任何动作都会暴露你
